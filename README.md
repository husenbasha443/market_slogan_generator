# Marketing Slogan Generator using Groq

## Objective
This project demonstrates prompt engineering by creating reusable prompt templates
to generate marketing slogans using Groq LLMs.

## Technologies Used
- Python
- Groq API (LLaMA 3)
- Prompt Engineering

## Project Structure
- prompts/prompt_library.py : Reusable prompt templates
- app.py : Demonstrates prompt usage and Groq API call
- requirements.txt : Dependencies

## Key Concepts
- Role-based prompting
- Task definition
- Constraints for controlled output
- Prompt reusability using variables
